You are given:
(1) a JSON list of capabilities (tools): each has name, required columns, required inputs, optional inputs, and script

- Do NOT invent columns or inputs that are not available.
- Prefer robust, safe defaults when uncertain.
- Explain your reasoning briefly.

Return valid JSON only, with this schema:
{
  "chosen_capability": "<name>",
  "args": { "csv": "...", "...": "..." },
  "reasons": ["...", "..."]
}
